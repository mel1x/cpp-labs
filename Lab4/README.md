# Задание 1. Класс QuadraticEquation

## Постановка задачи
Реализовать класс `QuadraticEquation` для работы с квадратными уравнениями вида ax^2 + bx + c = 0.
Класс должен содержать:
- Поля: `double a`, `double b`, `double c` (коэффициенты).
- Конструкторы: не менее трех (например, по умолчанию, с параметрами, копирования).
- Перегрузку оператора вывода `<<` для вывода полей класса.
- Метод вычисления корней уравнения. Результат должен быть массивом величин типа `double` (от 0 до 2-х элементов).

Протестировать все методы, включая конструкторы. Ввод данных с клавиатуры с проверкой корректности.

## Алгоритм решения
1. Создаю класс QuadraticEquation с полями a, b, c
2. Реализую конструкторы для инициализации полей
3. В методе solve вычисляю дискриминант по формуле b^2 - 4ac
4. Если дискриминант больше 0, возвращаю два корня
5. Если равен 0, возвращаю один корень
6. Если меньше 0, возвращаю пустой массив
7. Перегружаю оператор вывода для отображения уравнения

## Тесты
```cpp
QuadraticEquation eq1(1, -3, 2);
// Вывод: 1x^2 + -3x + 2 = 0
// Корни: 2, 1

QuadraticEquation eq2(1, 2, 1);
// Вывод: 1x^2 + 2x + 1 = 0
// Корни: -1

QuadraticEquation eq3(1, 0, 1);
// Вывод: 1x^2 + 0x + 1 = 0
// Корни: нет корней
```

# Задание 2. Перегрузка операторов

## Постановка задачи
Добавить к классу `QuadraticEquation` следующие перегруженные операции:

**Унарные операции:**
- `++`: увеличивает коэффициенты уравнения на 1.
- `--`: уменьшает коэффициенты уравнения на 1.

**Операции приведения типа:**
- `double` (неявная): результатом является дискриминант уравнения.
- `bool` (явная): результатом является `true`, если корни существуют D >= 0, иначе `false`.

**Бинарные операции:**
- `==`: уравнения равны, если равны их коэффициенты.
- `!=`: уравнения не равны, если не равны их коэффициенты.


## Алгоритм решения
1. В операторах ++ и -- увеличиваю или уменьшаю все коэффициенты на 1
2. В операторе приведения к double возвращаю значение дискриминанта
3. В операторе приведения к bool возвращаю true, если дискриминант неотрицательный
4. В операторах сравнения == и != проверяю равенство всех коэффициентов уравнений

## Тесты
```cpp
QuadraticEquation eq(1, 2, 1);

// Проверка операторов ++ и --
// Изначальное уравнение: 1x^2 + 2x + 1 = 0
// После ++: 2x^2 + 3x + 2 = 0
// После --: 1x^2 + 2x + 1 = 0

// Проверка преобразований
double d = eq; // Неявное к double (дискриминант) -> 0
bool hasRoots = static_cast<bool>(eq); // Явное к bool (D >= 0) -> true

// Проверка сравнений
QuadraticEquation eq2(1, 2, 1);
QuadraticEquation eq3(2, 2, 3);
// eq == eq2 -> true
// eq != eq3 -> true
```
